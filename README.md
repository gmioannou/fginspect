## Usage: fginspect filegdb

### Inspect ESRI File Geodatabase and create a report in the same folder.

### The report lists:

- `Datasets`
- `Feature classes`
- `Feature classes fields`
- `Tables`
- `Tables fields`
- `Domains`
- `Subtypes`
- `Relationships`
